Common problems and practical fixes

  • The app will not install: Download the file again, confirm the Android requirement, and allow installs from the app that opened the package.
  • The install opens but stops early: Free some storage and make sure the saved file is complete.
  • The app launches but feels broken: Restart the device, test the main feature again, and compare with an older or newer build if needed.
  • Updates keep failing: Remove the current build only if you have a clean replacement ready and know which version you are moving to.
